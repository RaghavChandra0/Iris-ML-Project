# Iris-ML-Project
 This project consists of Iris Data. It is based on Logistic Regression.
 #### Our Feautes consists of:-
<ol>
  <li>Sepal Length</li>
  <li>Sepal Width</li>
  <li>Petal Length</li>
  <li>Petal Width</li>
</ol>


#### Our target represents the type of Iris Flower:-
- 0 ----> Setosa
- 1 ----> Versicolor
- 2 ----> Verginica
